# References

Ce dossier contient toutes les références vers lesquelles ce projet pointe.
De cette manière, nous évitons d'avoir des chemins relatifs spécifiques à chaque contributeur dans le fichier `Properties/AssemblyInfo.cs`.

Si une référence manque, c'est dans ce dossier que vous la trouverez.
En l'absence d'une référence, merci de bien vouloir l'ajouter à ce dossier et de pointer vers celle-ci.

Veillez à la date d'ajout des références, car certaines peuvent nécessiter une mise à jour manuelle.
