# Config

Ce dossier contient les classes représentant la structure des fichiers de configuration pour faciliter la sérialisation/désérialisation en JSON.

## Structure

- Classes : Chaque classe doit représenter une partie spécifique de la configuration du plugin si celle-ci devient trop volumineuse.
- Propriétés : Les propriétés des classes doivent être précédées de commentaires descriptifs et suivre la convention PascalCase.
